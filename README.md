# faba pantranscriptome scripts

This repository contains the analysis scripts used in the faba pan-transcriptome paper.

These are organised as follows:

aminoacidcount.R: this script is used to count the amino acid content for each isoform.

stablegene_preferisoform.R: this script is used to calculate the SD in gene-level and find isoform switch in each gene.

WGCNA.R: this script is used to identify function module.

fig6.R: this script is used for figure 6 visualization.
